# 이 자판기 프로그램은 사용자가 음료를 선택하고, 선택한 음료의 개수에 따라 총 금액을 계산한 후, 돈을 투입하고 거스름돈을 출력합니다.

[메인 페이지로 돌아가기](https://github.com/jaeyong0311?tab=repositories)

<u>자판기 시뮬레이션 프로그램입니다</u>

**음료 선택
음료 개수 입력
총 금액 계산 및 출력
돈을 투입하고 거스름돈을 계산하여 출력**
